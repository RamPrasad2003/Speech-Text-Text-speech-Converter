Welcome to our Speech-to-Text and Text-to-Speech Recognition Tool :

Guidelines
  • First,select the type tool (Either speech-text or text-speech) in right side panel

    • Speech-Text
    • Here,you should have a well defined WAV audio file
    • After that,you have to upload that file
    • Next,click that translate button
    • Then you see the text in bottom of the translate button

    Text-Speech
    • Here you see a input field
    • Then, type the text that you want to translate to audio file
    • Next,click that translate button
    • Then you see the audio file in bottom of the translate button
